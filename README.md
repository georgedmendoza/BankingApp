# BankingApp

A minimal ASP.NET Core Web APi that simulates basic banking operations (currently deposit)
## Features

- 'POST /Banking/deposit' endpoint
- Model validation on incoming requests
- Simple service + repository layering
- In-memory storage for account balances (no db required)

## Tech Stack
- .Net 6
- C#
- ASP.NEt Core Web API
